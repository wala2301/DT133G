GitHub Enterprise is the self-hosted or cloud-managed version of GitHub designed for large organisations with requirements for advanced access control, compliance, and private networking.
GitHub Enterprise Cloud (GHEC) is the managed cloud offering. It adds enterprise-grade features to GitHub.com including SAML single sign-on, SCIM provisioning for automatically managing team membership from an identity provider, audit log streaming to SIEM systems, and IP allow lists to restrict access to known network ranges.
GitHub Enterprise Server (GHES) is the self-hosted option, installed on infrastructure managed by the organisation. GHES provides the same core GitHub features in an air-gapped or network-restricted environment, which is required by organisations with strict data residency or compliance requirements.
Enterprise accounts aggregate multiple organisations under a single billing and policy umbrella. Enterprise owners can enforce policies across all organisations, such as requiring 2FA, disabling public repository creation, and mandating specific authentication methods, without managing each organisation separately.
