Python's datetime module provides classes for representing and manipulating dates, times, and intervals.
The date class represents a calendar date with year, month, and day. datetime is a subclass that adds hour, minute, second, and microsecond fields. time represents a time of day independently of a date. timedelta represents the difference between two dates or datetimes and supports arithmetic with date and datetime objects.
datetime.now() returns the current local date and time. datetime.utcnow() returns the current UTC time without timezone information. For timezone-aware datetimes, datetime.now(tz=timezone.utc) is preferred because it returns an aware datetime that can be safely compared with other aware datetimes.
Formatting and parsing use strftime (format a datetime to a string) and strptime (parse a string into a datetime) with format codes such as %Y for four-digit year, %m for month, %d for day, %H for hour, and %M for minute. The isoformat() method produces ISO 8601 strings suitable for APIs and storage.
