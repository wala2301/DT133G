GitHub Codespaces provides cloud-hosted development environments that are preconfigured for a repository, accessible from a browser or VS Code.
A Codespace is a virtual machine running a Docker container defined by the repository's devcontainer.json configuration. The configuration specifies the base image, extensions to install in VS Code, port forwarding, environment variables, and setup scripts to run when the environment is created.
Developers can create a Codespace directly from a repository on GitHub, selecting the branch, region, and machine type. The environment is ready in seconds with all dependencies pre-installed, eliminating the need to set up a local development environment.
Codespaces are billed per core-hour of compute and per GB of storage. They are automatically stopped after a period of inactivity to reduce cost. The devcontainer.json specification is an open standard, meaning the same configuration works with VS Code's Dev Containers extension for local container-based development.
