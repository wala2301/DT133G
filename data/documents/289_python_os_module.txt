Python's os module provides functions for interacting with the operating system, covering file and directory operations, process management, and environment variables.
os.environ is a mapping of environment variable names to values. os.getenv(key, default) retrieves an environment variable safely, returning the default if it is not set. os.environ.get behaves identically. Environment variables should be read at startup and not cached in module-level variables if they may be changed at runtime.
File system operations include os.listdir(path) to list directory contents, os.makedirs(path, exist_ok=True) to create a directory tree, os.remove(path) to delete a file, and os.rename(src, dst) to rename or move a file. os.path provides utilities such as os.path.join, os.path.exists, os.path.isfile, and os.path.dirname.
os.getcwd() returns the current working directory and os.chdir(path) changes it. For most new code, pathlib.Path is preferred over os.path because it provides an object-oriented interface that is easier to read and compose, but os functions remain useful for process management and environment variable access.
