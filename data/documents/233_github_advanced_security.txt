GitHub Advanced Security (GHAS) is a suite of security features available for enterprise and organisation accounts on GitHub.
GHAS includes code scanning powered by CodeQL, secret scanning with push protection, and dependency review in pull requests. These features surface security issues earlier in the development lifecycle, before code reaches production.
For public repositories, code scanning and secret scanning are available at no extra cost. For private and internal repositories, GHAS requires a licence based on the number of unique active committers.
The security overview dashboard in GHAS-enabled organisations aggregates alerts across all repositories, allowing security teams to prioritise and track remediation efforts at scale. Alerts can be assigned to developers and tracked through to resolution.
