Python's itertools module provides memory-efficient tools for creating iterators that operate on iterables.
itertools.chain combines multiple iterables into a single iterator that yields items from the first iterable, then the second, and so on, without creating an intermediate list. itertools.islice slices an iterator by index without materialising all elements.
itertools.groupby groups consecutive elements of an iterable by a key function, yielding (key, group_iterator) pairs. The input must be sorted by the key for all same-key items to appear in the same group. itertools.product computes the Cartesian product of input iterables, equivalent to nested for loops.
itertools.combinations and itertools.permutations generate all combinations or permutations of a given length from an iterable. itertools.accumulate produces running totals or other accumulated results using a specified binary function. These tools compose well with generator expressions and avoid building large intermediate lists, making them efficient for processing large datasets.
