A JSON Web Token (JWT) is a compact, URL-safe token format used to represent claims
between two parties. A JWT consists of three Base64URL-encoded parts separated by dots:
a header describing the signing algorithm, a payload containing claims, and a signature.
Standard claims include iss (issuer), sub (subject), exp (expiration time), and iat
(issued at). JWTs are commonly used as bearer tokens in OAuth 2.0 and OpenID Connect,
where the resource server validates the signature to verify authenticity without
contacting the authorization server for each request.
