The Python collections module provides specialised container types that extend the built-in list, dict, set, and tuple with additional functionality.
Counter is a subclass of dict designed for counting hashable objects. Calling Counter(iterable) returns a mapping of items to their occurrence counts. The most_common(n) method returns the n most frequent elements.
defaultdict is a dict subclass that calls a factory function to supply missing values rather than raising KeyError. Passing list as the factory creates a dictionary where missing keys automatically initialise to an empty list.
OrderedDict preserves insertion order (redundant in Python 3.7+ dicts but still useful for reordering operations) and deque provides a double-ended queue with O(1) appends and pops from both ends, making it efficient for implementing queues and sliding windows. namedtuple creates tuple subclasses with named fields, combining the immutability of tuples with the readability of named attributes.
